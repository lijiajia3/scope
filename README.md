# SCOPE: Label-free Self-Coverage Certification for Black-box LLM Information Extraction

Code, data, and reproduction scripts for the paper
**"SCOPE: Label-free self-coverage certification for black-box large language model information extraction"** (Guo & Li).

SCOPE audits, without any gold labels, how completely a stochastic black-box LLM
extractor has covered a sentence. It decodes each sentence *K* times, treats each
distinct entity as a species with a cross-decode incidence count, and applies
Good–Turing / Chao–Jost missing-mass estimation to obtain a per-sentence
**self-coverage certificate** `Ĉ = 1 − f₁/N`. It then **certifies** stable
sentences, **abstains** when decoding collapses, and ranks sentences by risk
`1 − Ĉ` for **active-annotation triage**.

## Data

- `data/clinical_zh_gold_520.json` — **Clinical-ZH**: 520 Chinese medical/pharmacological
  statements with gold NER annotations. These are general medical statements drawn
  from published medical literature and drug information (e.g. drug interactions,
  standard dosages); they contain **no patient records and no personally
  identifiable information**, verified before release. Not human-subjects research.
- **CoNLL-EN** is loaded from the public `tomaarsen/conll2003` release via
  `datasets`; it is not redistributed here.

## Reproducing the paper

```bash
pip install -r requirements.txt        # torch/transformers/openai/datasets/matplotlib
export SILICONFLOW_API_KEY=...          # or place the key in ~/.siliconflow_key
```

Main pipeline (decode → estimate → evaluate):

```bash
python3 full_experiment.py --dataset zh --n 520 --K 20 --tag 7b --model Qwen/Qwen2.5-7B-Instruct
python3 full_experiment.py --dataset en --n 500 --K 20 --tag 7b --model Qwen/Qwen2.5-7B-Instruct
python3 estimate_v2.py                  # Good-Turing/Chao self-coverage + variant merging
python3 evaluate_v2.py                  # calibration / certify-or-abstain / decomposition / triage
```

Analyses in the paper:

| Script | Produces |
|---|---|
| `ablation_naive.py` | naive-count vs. self-coverage abstention (Table 2) |
| `mechanism_auroc.py` + `fig_mechanism.py` | why entropy/consistency baselines fall below chance |
| `eval_all_tags.py` | all metrics with bootstrap 95% CI (temperature, repeats, full-set, cross-family) |
| `analyze_k100.py` | K=100 deep-sampling of the omission share |
| `eval_triage_baselines.py` | triage AUROC incl. verbalized-confidence baseline |
| `threshold_sensitivity.py` | abstention threshold 2D sensitivity sweep |
| `triage_significance.py` | bootstrap confidence intervals on triage AUROC gaps |
| `exact_match_eval.py` | exact-match precision/recall/F1 vs. containment criterion |
| `crossfamily_eval.py` | cross-model-family metrics (Table 3) |

Every number and table in the paper is regenerated by these scripts from
the decode sets in `out/`.

## Cross-family experiments

Non-Qwen decode sets are in `out/full_decodes_{ds}_{family}.jsonl`. Run:

```bash
python3 full_experiment.py --dataset en --tag glm --model THUDM/GLM-4-9B-0414
python3 full_experiment.py --dataset en --tag dsv3 --model deepseek-ai/DeepSeek-V3
python3 full_experiment.py --dataset en --tag ling --model inclusionAI/Ling-mini-2.0
python3 full_experiment.py --dataset en --tag hunyuan --model tencent/Hunyuan-A13B-Instruct
```

Other backends: append `--base_url` for ollama, deepinfra, together, or groq.

## Layout

- `full_experiment.py`, `estimate_v2.py`, `evaluate_v2.py` — main pipeline
- `ablation_naive.py`, `eval_all_tags.py`, `analyze_k100.py`, `crossfamily_eval.py`, etc. — analyses
- `config.py` — shared paths and prompts
- `out/` — decode sets, per-model estimates, computed metrics
- `data/` — Clinical-ZH gold corpus
